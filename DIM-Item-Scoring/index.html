Item-Scoring Extension for Destiny Item Manager

About

This is a browser extension designed to work with the Compare Items UI in Destiny Item Manager. Its purpose is to make it easier to figure out which weapon is "best" according to a formula applied to each weapon's stats. Learn more.

Installation

This browser extension will be available in the Chrome Web Store.

Usage
  1. Install this extension
  2. Open and sign into Destiny Item Manager.
  3. View your Inventory.
  4. Use the "Compare Items" UI to compare two or more weapons.
  5. A new and sortable row of numbers will automatically appear between the weapon names and Power stat. See the sceenshot above.
  6. Each number is the weapon's "score", and updates automatically as you add and remove weapons from the Compare Items UI.
  7. Learn more about how the extension works, including some tips and tricks.
Known Issues
  • Can't sort items by their score.
  • Can't customize the formulas used to calculate the weapon scores.
  • No scores for armor.
  • Power doesn't affect weapon scores.
  • Some perks don't affect weapon scores.
  • Sorting items, adding items, and removing items can scramble the Comprae Items table.
